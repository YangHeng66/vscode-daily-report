# Change Log

All notable changes to the "Daily Report Generator" extension will be documented in this file.

## [0.0.1] - 2024-12-01

### Added
- Initial release
- Generate daily reports from Git commit history
- Generate weekly reports from Git commit history
- Generate custom date range reports
- AI-powered report summarization (DeepSeek, OpenAI, Anthropic)
- Simple report generation without AI
- Support for Chinese and English reports
- Configurable author filter
- Auto-detect Git repositories
